Reading guide:
  - Entry is what users pay before hydration. This is the metric to drive down.
  - Route-specific entry is the portion unique to this route (Entry minus shared).
  - All client includes async chunks (next/dynamic, lazy()). For the wrap pattern in
    server components with ssr:true, async chunks are typically still in Entry —
    deferral requires ssr:false in a client component shim.
